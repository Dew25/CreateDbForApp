Программа CreateDbForApp предназначена для быстрого создания базы данных используемой в приложении.
Как это работает.
1. В файле database.properties запишите название (имя) вашей базы данных, имя пользователя базы данных и пароль доступа этого пользователя (обычно, все три параметра для учебной программы совпадают, для других случаев обязательно укжите индивидуальный сложный пароль).
2. Запустите двойным кликом пакетный файл Start. (если не работает, то ищите решение запустить CreateDbForApp.jar с библиотекой lib для вашей системы. Обычно надо выполнить в cmd (PowerShell) следующую команду: 
java -cp lib/* -jar CreateDbForApp.jar
Укажите путь к исполняемому файлу java.exe на вашем компьютере, если он не прописан в переменной окружения Path)
Пользователь, пароль и база данных будут созданы и настроены на использование в Java Web приложении (База будет настроена на использование utf8_general_ci, a указанному пользователю будут предоставлены все привилегии управления этой базой).
Примечание: У вас должен быть установлен XAMPP с базовыми настойками. В противном случае, слушайте подсказки программы и выполняйте ее просьбы.
Если программа не выполняет свою работу, проверьте наличие такой базы данных или пользователя. Удалите существующую базу и пользователя. Запустите программу снова.
